# Benchmark Results

This directory stores benchmark execution results.

## Files

- `benchmark_suite_TIMESTAMP.json` - Complete benchmark suite results
- `.gitkeep` - Ensures directory is tracked in git

## Running Benchmarks

```bash
python tests/benchmark_suite.py
```

Results are automatically saved with timestamp.

## Result Format

See `tests/BENCHMARKS_README.md` for JSON schema and expected values.
